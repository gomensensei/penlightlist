# 🌸 AKB48 成員名單及應援色生成器  
## AKB48 ペンライトカラー生成ツール  
## AKB48 Penlight Generator

---

# 📖 簡介 | 概要 | Overview

**繁體中文**  
這是一個專為 AKB48 粉絲設計的視覺生成工具，用於快速製作成員名單及應援色（推しカラー）圖卡，適用於生誕祭、公演及演唱會應援。

**日本語**  
本ツールはAKB48ファンのために設計されたビジュアル生成ツールです。メンバー一覧とペンライトカラーを高品質な画像として簡単に作成できます。

**English**  
A visual generator designed for AKB48 fans to quickly create high-quality member lineup and penlight color graphics for events, concerts, and fan projects.

---

# ✨ 核心功能 | 主な機能 | Features

## 🎨 應援色呈現 | カラー表示 | Color Rendering
- 色塊方格（打斜 AB 色）
- 陰影色文字（Shadow Text）
- 精確還原推しカラー

---

## 📱 響應式佈局 | レスポンシブUI | Responsive Layout
- 電腦版：預覽 + 控制面板
- 手機版：上下結構設計

---

## ⚖️ 自動排版 | 自動レイアウト | Auto Layout
- 根據人數自動生成 Grid
- 支援特殊人數置中
- 一鍵載入期別名單

---

## 🌍 多語言 | 多言語対応 | Multi-language
- 繁體中文 / 简体中文 / 日本語 / English  
- 한국어 / ไทย / Bahasa Indonesia

---

## 🖋️ 文字處理 | テキスト処理 | Smart Text
- 自動縮放避免溢出
- 支援 Furigana

---

## 💾 高畫質輸出 | 高解像度出力 | High Quality Export
- PNG 無損輸出
- 適用印刷 / SNS / 應援

---

# 📁 檔案結構 | ファイル構成 | File Structure
